
# Interview Answers

## Purpose
This file contains sample stakeholder interview answers for the Student Task Management System. Each answer has an ID so it can be referenced by elicitation findings and requirements.

## Lecturer Interview Answers

### INT-01
**Question:** What problem do you face when managing assignments manually?

**Answer:** It is difficult to track which students have submitted, which students are missing, and which submissions are late when information is spread across different tools.

### INT-02
**Question:** What information should be available when viewing submissions?

**Answer:** I need to see student name, assignment name, submission time, file link, submission status, grade, and feedback status.

### INT-03
**Question:** What do you need when creating an assignment?

**Answer:** I need to enter the title, description, course, deadline, and submission instructions so students understand what to submit.

### INT-04
**Question:** What do you need when grading?

**Answer:** I need to open the submitted file, enter a grade, and write feedback for the student.

## Student Interview Answers

### INT-05
**Question:** What helps you manage assignment deadlines?

**Answer:** I need a dashboard that shows active assignments and upcoming deadlines for my courses.

### INT-06
**Question:** What do you expect after submitting a file?

**Answer:** I want a clear confirmation message because I need to know that my file was uploaded successfully.

### INT-07
**Question:** What assignment status information is useful?

**Answer:** I want to know whether an assignment is not submitted, submitted, late, graded, or returned.

### INT-08
**Question:** What do you need after grading is complete?

**Answer:** I need to see my grade and feedback from the lecturer.

## Administrator Interview Answers

### INT-09
**Question:** What data should administrators manage?

**Answer:** Administrators need to manage lecturer accounts, student accounts, courses, and enrollments so the assignment access is correct.

### INT-10
**Question:** What configuration is needed for assignment submissions?

**Answer:** Administrators need to set accepted file types, maximum file size, and late submission policy.

### INT-11
**Question:** What report is useful for administrators?

**Answer:** Administrators need reports that show assignment count, submitted count, missing count, and late submission count by course.

## Interview Notes
- These answers are used as evidence for elicitation findings.
- Each elicitation finding should reference the related interview ID when applicable.
- Interview answers should not be treated as final requirements until reviewed.
